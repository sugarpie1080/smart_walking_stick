Detailed Description

DDSCommunicator class for publishing data to DDS.

Author
Felicity Lipscomb
Version
0.1
Date
2025-02-25

This class is responsible for publishing data from the ToFSensor and MotorMove classes to DDS. It implements the CallbackInterface for both classes and uses a separate thread to handle the publishing. This is then used by the Webapp to display the data.

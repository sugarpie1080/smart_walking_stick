Detailed Description

Class for moving the motor based on the commands. Derived from CallbackInterface for the ToFSensor class.

This class is responsible for moving the motor based on the distance from the ToFSensor. It uses a separate thread to handle the motor movement and a callback interface to communicate with the ToFSensor.
